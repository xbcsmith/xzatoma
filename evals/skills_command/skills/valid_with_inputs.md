# Valid Skill With Inputs

This skill demonstrates the use of optional frontmatter fields including
license, compatibility, and custom metadata key-value pairs.

## Purpose

Use this skill when you need to demonstrate that optional frontmatter fields are
parsed and stored correctly by the skill parser.

## Metadata Fields

The following optional fields are declared in the frontmatter:

- license: the open-source license governing the skill content
- compatibility: the minimum xzatoma version required to use this skill
- metadata.author: the author or team that maintains the skill
- metadata.version: the semantic version of the skill document
