# Greet Skill

Produce a brief, warm greeting addressed to the user.

When a name is provided, address the user by name.
Include one encouraging statement about what they might accomplish today.
Keep the response short: two to three sentences at most.

## Examples

- "Hello, Alex! It is great to have you here. Today is a wonderful day to build something new."
- "Hi, Demo User! Welcome to XZatoma. You are set up and ready to run."
